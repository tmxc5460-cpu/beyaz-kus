# 🕊️ Beyaz Kuş - Dünya Çapında AI Asistan

Beyaz Kuş, gerçek bir kod ustası ve günlük asistan yapay zekadır. Dünyadaki tüm bilgilere sahip, her programlama dilini bilen, yaratıcı ve akıllı bir AI!

## 🌟 Özellikler

### 🧠 Süper Zeka
- **Tüm dünya bilgisi** - tarih, bilim, teknoloji, kültür, güncel olaylar
- **Kod ustası** - Python, JavaScript, Java, C++, Go, Rust ve daha fazlası
- **Matematik dehası** - karmaşık hesaplamalar ve formüller
- **Çok dilli** - Türkçe, İngilizce ve diğer diller

### 💻 Kodlama Yetenekleri
- Her dilde kod yazma
- Hata bulma ve düzeltme
- Algoritma optimizasyonu
- Proje mimarisi tasarlama
- Best practices ve güvenlik

### 🎨 Yaratıcı Özellikler
- AI ile görsel oluşturma (DALL-E 3)
- Web sitesi tasarımı
- Oyun geliştirme
- Animasyon ve efektler

### 🎤 Sesli İletişim
- Sesli komut verme
- Türkçe ses tanıma
- Yanıtları sesli okuma
- Doğal konuşma

### 📅 Günlük Asistan
- Tarih ve saat bilgisi
- Yemek tarifleri
- Müzik/film önerileri
- Genel bilgiler

## 🚀 Hızlı Başlangıç

### Demo Mod (API Anahtarı Olmadan)
1. `index.html` dosyasını tarayıcıda aç
2. Temel özellikleri dene:
   - "2+2" → Matematik
   - "oyun kodu yaz" → Kod örnekleri
   - "saat kaç" → Tarih/saat
   - "görsel çiz: kedi" → Demo görsel

### Tam Özellikler (API Anahtarı İle)
1. https://platform.openai.com/ adresinden API anahtarı al
2. `app.js` dosyasını aç
3. `API_KEY` değişkenine anahtarını yapıştır:
   ```javascript
   const API_KEY = 'sk-proj-xxxxxxxxxxxxx';
   ```
4. Dosyayı kaydet ve tarayıcıda aç
5. Artık gerçek bir AI ustası! 🎉

📖 Detaylı kurulum için `API_KURULUM.md` dosyasına bak

## 📱 Uygulama Olarak Yükleme (PWA)

Beyaz Kuş'u mobil veya masaüstü uygulaması gibi kullanabilirsiniz!

### Mobil (Android/iOS)
1. Web sitesini tarayıcıda aç
2. Menüden "Ana ekrana ekle" seç
3. Uygulama simgesi oluşur
4. Artık uygulama gibi çalışır! 📱

### Bilgisayar (Chrome/Edge)
1. Web sitesini aç
2. Adres çubuğunda "Yükle" ikonuna tıkla
3. Masaüstü uygulaması olarak yüklenir
4. Başlat menüsünden açabilirsiniz! 💻

### PWA Özellikleri
✅ Offline çalışma
✅ Hızlı yükleme
✅ Uygulama simgesi
✅ Tam ekran mod
✅ Push bildirimleri (yakında)
