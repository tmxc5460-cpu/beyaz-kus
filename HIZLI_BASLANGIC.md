# ⚡ Beyaz Kuş - Hızlı Başlangıç

5 dakikada web'e yayınla, herkes kullansın!

## 🎯 Hedef

Beyaz Kuş'u internete yayınlayıp, herkesin mobil ve bilgisayardan uygulama olarak yükleyebilmesini sağlamak.

## 📋 Yapılacaklar Listesi

### 1. İkonları Oluştur (2 dakika)
- [ ] `create-icons.html` dosyasını tarayıcıda aç
- [ ] `icon-192.png` dosyasını indir
- [ ] `icon-512.png` dosyasını indir
- [ ] İki dosyayı proje klasörüne kopyala

### 2. GitHub Hesabı (3 dakika)
- [ ] https://github.com adresine git
- [ ] "Sign up" ile hesap oluştur
- [ ] Email'ini doğrula

### 3. Repository Oluştur (1 dakika)
- [ ] GitHub'da "+" → "New repository"
- [ ] İsim: `beyazkus`
- [ ] Public seç
- [ ] "Create repository" tıkla

### 4. Dosyaları Yükle (3 dakika)
- [ ] "uploading an existing file" linkine tıkla
- [ ] Şu dosyaları sürükle-bırak:
  - `index.html`
  - `login.html`
  - `app.js`
  - `style.css`
  - `manifest.json`
  - `sw.js`
  - `icon-192.png`
  - `icon-512.png`
- [ ] "Commit changes" tıkla

### 5. GitHub Pages Aktif Et (2 dakika)
- [ ] Repository'de "Settings" tıkla
- [ ] Sol menüden "Pages" seç
- [ ] "Branch" → "main" seç
- [ ] "Save" tıkla
- [ ] 2-3 dakika bekle

### 6. Test Et! (1 dakika)
- [ ] `https://KULLANICI_ADIN.github.io/beyazkus/` adresini aç
- [ ] Site çalışıyor mu kontrol et
- [ ] Mobil/bilgisayarda "Yükle" butonunu dene

## ✅ Tamamlandı!

Artık herkes şu adresten erişebilir:
```
https://KULLANICI_ADIN.github.io/beyazkus/
```

## 📱 Kullanıcılar Nasıl Yükler?

### Mobil
1. Linki aç
2. Tarayıcı menüsü → "Ana ekrana ekle"
3. Uygulama simgesi oluşur

### Bilgisayar
1. Linki aç
2. Adres çubuğunda "Yükle" ikonuna tıkla
3. Masaüstü uygulaması olur

## 🔧 Sorun mu Var?

### Site açılmıyor
- 5 dakika bekle
- Tarayıcı cache'ini temizle (Ctrl+Shift+Delete)
- Gizli pencerede dene

### Uygulama yüklenmiyor
- İkon dosyalarını kontrol et
- HTTPS üzerinden eriştiğinden emin ol
- Chrome/Edge kullan

### Başka sorun
- [GITHUB_YAYINLAMA.md](GITHUB_YAYINLAMA.md) dosyasına bak
- GitHub Issues'da sor
